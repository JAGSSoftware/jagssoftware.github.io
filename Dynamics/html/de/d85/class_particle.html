§ initializeVelocity()

void Particle::initializeVelocity ( )
private

Set the initial velocity of the particle.

From the reduced temperature \(\tilde{T}\) it is calculated randomly all of the components of the velocity, having as squared module the value of the reduced temperature.

In spherical coordinates, \(0 \le \theta \le \pi, 0 \le \phi \le 2\pi \) are generated randomly, keeping the radius of the sphere constant (with value \(r = \sqrt{\tilde{T}}\))

§ temperature

double Particle::temperature
private

Kinetic energy of the particle as temperature.

The temperature is in reduced units: \(\tilde{T}=\frac{k_B T}{m}\)

The relation between velocity and temperature is:

\[ \frac{1}{2}mv_{rms}^2 = \frac{3}{2}k_BT \Rightarrow v_{rms}^2 = \frac{3k_BT}{m} = \tilde{T} \]
