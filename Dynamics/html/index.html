This is the implementation of the simple dynamic of a particle in a 3d geometrical inertial space, i.e. under no forces. The position is set in a random point in the space, and the initial velocity is calculated from Temperature, setting randomly the coordinates for the velocity in the space.

The relation between Temperature \(T\) and velocity \(v\) is through the expression:

\[ E_k = \frac{1}{2}mv^2=\frac{3}{2}k_BT \rightarrow v^2 = \frac{3k_B}{m}T \]

To set the coordinates of the velocity, spherical coordinates are used:

\[ \array{ v_x = |v| \sin{\theta} \cos{\phi} \cr v_y = |v| \sin{\theta} \sin{\phi} \cr v_z = |v| \cos{\theta} } \]

where \( |v| = \sqrt{T}, 0 \le \theta \le \pi \) and \( 0 \le \phi \le 2\pi \), \( \theta \) and \( \phi \) are taken randomly.
